# Vercel Setup Guide

Guía rápida para configurar Vercel para Calixo PWA.

## Quick Start

1. **Instalar Vercel CLI:**
   ```bash
   npm i -g vercel
   ```

2. **Login:**
   ```bash
   vercel login
   ```

3. **Link proyecto:**
   ```bash
   vercel link
   ```

4. **Deploy preview:**
   ```bash
   vercel
   ```

5. **Deploy production:**
   ```bash
   vercel --prod
   ```

## Configuración en Dashboard

1. Ve a [vercel.com](https://vercel.com)
2. Importa repositorio GitHub
3. Configura variables de entorno
4. Deploy automático en cada push a `main`
